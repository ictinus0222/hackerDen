# Product Overview

hackerDen is a focused MVP hackathon management tool designed to streamline team workflows from formation to submission. The product addresses the natural phases of hackathons: kick-off chaos, intensive development sprint, and final submission crunch.

## Core Features

- **Project Hub**: Centralized project information and team management
- **Task Board**: Visual Kanban-style task management with drag-and-drop
- **Pivot Tracking**: Document and track major direction changes
- **Submission Package**: Generate clean submission pages for judges
- **Real-time Collaboration**: Live updates across all team members
- **Mobile Responsive**: Works on phones and tablets

## Target Users

Hackathon teams (1-6 members) who need coordination during fast-paced development events. The tool eliminates "multi-tool disease" by providing phase-specific features that align with typical hackathon workflows.

## Key Value Propositions

1. Single source of truth for project information
2. Real-time collaboration without coordination overhead
3. Automated submission package generation
4. Mobile-first responsive design for venue mobility
5. Focused feature set that prevents tool sprawl
